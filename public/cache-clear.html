🔧 Limpeza de Cache

Use esta página para resolver problemas de cache em dispositivos móveis.

📋 Instruções para Problemas de Acesso:

  1. Clique em "Limpar Todo Cache"
  2. Clique em "Limpar Storage Local"
  3. Feche completamente o navegador
  4. Abra novamente e acesse o app
  5. Se ainda não funcionar, reinicie o celular
